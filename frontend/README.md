## Installation
* first thing to do is to install all dependencies  
```bash
npm install && bower install
```

* next you can run  
```bash
grunt && grunt serve
```

* If you do not have node and bower installed please refer to [yeoman setup guide](http://yeoman.io/codelab/setup.html)
